Descriere Proiect

Acest proiect este un proiect realizat pentru materia Tehnologii Web, in care sunt folosite urmatoarele tehnologii:

  • HTML  
  • CSS  
  • Javascript  

Pentru a vizualiza proiectul faceti click pe imaginea de sub:

HTML

HTML inseamna Hyper Text Markup Language

  1. HTML este un limbaj de editare. Ce inseamna asta? Inseamna ca:
    • Nu poti programa cu el
    • Nu poti face calcule matematice
    • Poti afisa informatii intr-un site
    • Poti introduce tabele intr-un site

  2. HTML-ul nu este pentru design, desi a fost o perioada in care HTML-ul si design-ul mergeau mana in mana.

  3. Pentru design avem un alt limbaj si anume CSS.


CSS

CSS inseamna Cascading Style Sheets

  1. CSS este un limbaj de stilizare al elementelor HTML

  2. In web deignul modern, pentru stilizarea paginilor se foloseste numai CSS

  3. Exista mai multe feluri de CSS
    • Incorporat in pagina
    • CSS care se afla in fisiere externe si sunt "chemate" in fisierul html
    • Incorporat in tagurile HTML

  4. Pentru folosirea CSS trebuie sa cunoasteti limbajul HTML



Javascript

JavaScript este un limbaj de scripting client-side care ruleaza la nivelul browserului

  1. Javascript ofera designerilor HTML posibilitatea executarii scripturilor la nivelul browserului

  2. Este un limbaj interpretat

  3. Javascript poate citi si scrie elemente HTML

  4. Poate fi folosit pentru a valida datele din formulare inainte de a fi trimise catre server

  5. Este folosi pentru a adauga dinamism paginilor
